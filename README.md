# NN Fund Management

A custom Odoo 19 Community module developed for the NN Services & Engineering Ltd. Trainee Software Developer technical assessment.

## Module

`nn_fund_management`

## Development Status

Active development.

## Planned Scope

- Fund accounts and incoming funds
- Project and expense-head allocations
- GM and MD approval workflows
- Fund requisitions
- Controlled bill posting and cancellation
- Fund transfers
- Audit and approval history
- Multi-company security
- Automated tests
- Dockerized deployment

## Current Environment

- Odoo 19 Community
- Python 3.12
- PostgreSQL 15
- Windows 11 local development environment

Detailed installation, testing, architecture, assumptions, limitations, AI-use disclosure, and demonstration instructions will be added as development progresses.
